使用Hexo和GitHub搭建博客

发表于 2014-11-05   |  

本文只是一篇总结性的文章，详细搭建教程可参照 zipperary、poem_of_sunshine的专栏。

安装Git

下载msysgit执行安装即可。

安装Node.js

下载Node.js执行安装即可。

安装hexo

使用Git Bash输入一下命令安装hexo：

$ npm install -g hexo

初始化hexo

创建一个名为 Blog 的件夹，然后在此文件中右键打开 Git Bash，输入以下的命令：

$ hexo init

安装依赖包

$ npm install

发布到本地

执行这条命令后，Blog 文件夹会自动生成需要的所有文件，再依次执行以下两个命令即可用浏览器访问http://localhost:4000/。

$ hexo generate
$ hexo server

到此，本地的博客已搭建完毕，接下来就是把本地的hexo博客部署到Github。

创建repository

注册Github帐号，然后以自己的帐号名创建一个新的repository，例如：name.github.io（name换成你的帐号）。

部署到Github

将Blog文件夹下的_config.yml作如下修改（name换成你的帐号）：

deploy:
  type: github
  repository: https://github.com/name/name.github.io.git
  branch: master

修改完后执行如下命令：

$ hexo generate
$ hexo deploy

此时你的hexo博客已经大功告成，用浏览器访问http://name.github.io/。

写博客

建博客的目的当然是为了写博客，执行如下命令：

$ hexo new "first-blog"

执行完后，打开Blog/source_posts/first-blog.md，然后尽情的书写吧！

写完博客后别忘了重新部署到GitHub上：

$ hexo generate
$ hexo deploy

当然也可以先本地预览：

$ hexo generate
$ hexo server

简写

$ hexo generate
$ hexo deploy
$ hexo server
$ hexo new "first-blog"

以上命令可如下简写：

$ hexo g
$ hexo d
$ hexo s
$ hexo n "first-blog"
